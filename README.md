This repo is to use nnstreamer and nnstreamer-pytorch in docker on Jetson AGX Orin and Jetson Orin nano.  
I have submitted the docker image on dockerhub. If you just want to use it instead of building it from scratch, you can simply run
```
docker pull liuhao97/nnstreamer_jetson:latest
```
to pull the image.  
Then enter the docker, and enter the commands in the block of ``After docker commit, how to resume``.

1. https://github.com/nnstreamer/nnstreamer
2. https://nnstreamer.github.io/getting-started.html
